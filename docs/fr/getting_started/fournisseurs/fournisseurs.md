# Fournisseurs

![Panneau de gestion des fournisseurs](./screenshots/image1.png)

## Créer un fournisseur

- Cliquer sur `Ajouter un fournisseur` (bouton en haut à droite)
  ![Ajouter un fournisseur](./screenshots/image2.png)
- Saisir les informations
  ![Saisir les informations](./screenshots/image3.png)

- Cliquer sur `Enregistrer`

## Gestion des modèles

- Cliquer sur `Modèles`
  ![Cliquer sur `Modèles](./screenshots/image4.png)
- Rentrer les informations:
  ![Modèle](./screenshots/image5.png)
  - `Nom proxy` est le nom qui sera visible par les utilisateurs sur les différents outils utilisant Xolo
  - `Modèle réel (transmis au fournisseur)` est le nom du modèle chez le fournisseur
  - `Params actifs (Md)`, `Débit min (tok/s)`, `Débit max (tok/s)` sont des valeurs utilisées pour le calcul des données
  - TODO FAIRE LES INFOS DU RESTE ?
- Cliquer sur `Enregistrer`

## Supprimer un modèle

- Cliquer sur `Modifier`
  ![Cliquer sur `Modifier`](./screenshots/image6.png)
- Cliquer sur `Supprimer` (bouton en bas à droite)
  ![Cliquer sur `Supprimer`](./screenshots/image7.png)
